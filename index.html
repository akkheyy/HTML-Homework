Summary: Latitude Analysis


The purpose of this site is to analyze how weather changes near the equator. To accomplish this analysis, data was used from the OpenWeatherMap API to assemble a dataset of over 500 cities.


After assembling the dataset, Matplotlib was used to plot various aspects of weather vs. latitude. Factors plotted include: cloudiness, humidity, temperature, and wind speed. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations.
